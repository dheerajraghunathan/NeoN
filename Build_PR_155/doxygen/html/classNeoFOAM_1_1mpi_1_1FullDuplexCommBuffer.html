Detailed Description

A buffer for full-duplex communication in a distributed system using MPI.

The FullDuplexCommBuffer class facilitates efficient, non-blocking, point-to-point data exchange between MPI ranks, allowing for simultaneous send and receive operations. It manages two HalfDuplexCommBuffer instances: one for sending data and one for receiving data.
