\doxysubsection{Detailed Description}
A buffer for full-\/duplex communication in a distributed system using MPI. 

The \mbox{\hyperlink{classNeoFOAM_1_1mpi_1_1FullDuplexCommBuffer}{Full\+Duplex\+Comm\+Buffer}} class facilitates efficient, non-\/blocking, point-\/to-\/point data exchange between MPI ranks, allowing for simultaneous send and receive operations. It manages two \mbox{\hyperlink{classNeoFOAM_1_1mpi_1_1HalfDuplexCommBuffer}{Half\+Duplex\+Comm\+Buffer}} instances\+: one for sending data and one for receiving data.
