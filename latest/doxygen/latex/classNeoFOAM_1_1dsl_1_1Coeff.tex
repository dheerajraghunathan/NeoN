\doxysubsection{Detailed Description}
A class that represents a coefficient for the \doxylink{namespaceNeoFOAM}{Neo\+FOAM} dsl. 

This class stores a single scalar coefficient and optionally span of values. It is used to delay the evaluation of a scalar multiplication with a field to avoid the creation of a temporary field copy. It provides an indexing operator {\ttfamily operator\mbox{[}\mbox{]}} that returns the evaluated value at the specified index.
