class Neo\+N\+::\+Domain\+Field$<$ Value\+Type $>$}
Represents the domain fields for a computational domain. 

The \doxylink{classNeoN_1_1DomainField}{Domain\+Field} class stores the internal fields and boundary information for a computational domain. It provides access to the computed values, reference values, value fractions, reference gradients, boundary types, offsets, and the number of boundaries and boundary faces.
