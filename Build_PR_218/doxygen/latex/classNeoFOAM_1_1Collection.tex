\doxysubsection{Detailed Description}
A type-\/erased interface collection types. 

The \doxylink{classNeoFOAM_1_1Collection}{Collection} class provides a type-\/erased interface for different collection types. It allows operations such as retrieving documents, finding documents based on predicates, and accessing collection metadata.
