\subsubsection*{template$<$typename Value\+Type$>$\newline
class Neo\+FOAM\+::\+Domain\+Field$<$ Value\+Type $>$}
Represents a field that expands over the full computational domain ie the internal field including boundary data. 

The \doxylink{classNeoFOAM_1_1DomainField}{Domain\+Field} class stores the internal field and the field data on the boundary for a computational domain. It provides access to the computed values, reference values, value fractions, reference gradients, boundary types, offsets, and the number of boundaries and boundary faces.
