Detailed Description

template<typename ValueType>
class NeoFOAM::DomainField< ValueType >

Represents a field that expands over the full computational domain ie the internal field including boundary data.

The DomainField class stores the internal field and the field data on the boundary for a computational domain. It provides access to the computed values, reference values, value fractions, reference gradients, boundary types, offsets, and the number of boundaries and boundary faces.
